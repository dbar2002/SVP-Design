# Images Directory

Place your logo files here:

- `logo.png` — Main logo (recommended: 200×60px or similar, transparent PNG)
- `logo-light.png` — Light/white version for dark backgrounds (optional, falls back to logo.png)
- `favicon.ico` — Favicon (32×32px or 16×16px .ico file)
- `favicon.png` — Favicon PNG (32×32px, used as fallback)
- `logo-icon.png` — Square icon version for small spaces (optional, 48×48px)

The site references these filenames throughout all pages.
